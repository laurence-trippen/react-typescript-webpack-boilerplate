# React TypeScript Webpack5 Boilerplate

by Laurence Trippen

### Features

- [x] TypeScript with extra Type-Checking step (not only transpiling...)
- [x] Babel
- [x] Jest
- [ ] React Testing Library
- [ ] ESLint?
- [ ] Prettier?
- [ ] Cypress?
- [ ] Storybook?

### Usage

```bash
npm i

# for development
npm run dev

# for production
npm run build

# for Jest unit-testing
npm test
```
